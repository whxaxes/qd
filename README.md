# Create pagelet or vue project page directory quickly

## Install

```bash
npm install quick-dir -g
```

## Use

```bash
qd test
```

## Directory structure

vue 

```
└── test
    ├── index.scss
    ├── index.vue
    └── vm.js
```

pagelet

```
└── test
    ├── test.js
    ├── test.scss
    └── test.tpl
```
